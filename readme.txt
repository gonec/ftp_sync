Синхронизируем два фтп с записью в базу данных
База данных расположена на другом сервере, т.к. на
локальном установить не удалось

1. Для настройки программы необходимо прописать в cron файле путь к скрипту 
sync.py 
например
* * * * * /usr/local/bin/python3 /home/user/ftp_sync/sync.py

после чего cron файл поместить в папку /etc/cron.d/
